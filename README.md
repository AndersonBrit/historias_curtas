Projeto web simples que apresenta **histórias curtas interativas** em páginas HTML.  
O utilizador pode navegar pelo menu para escolher uma história e voltar ao menu principal quando quiser.

---

## 🗂 Estrutura do projeto

```
historias_curtas/
│
├─ assets/                            → Imagens
│
├─ src/
│ ├─ Historia_curta-menu.html
│ ├─ Historia_curta-historia_1.html
│ └─ Historia_curta-historia_2.html
│
└─ README.md
```


---

## 📖 Como funciona

1. Abra `Historia_curta-menu.html` no navegador.  
2. Clique em **Historia 1** ou **Historia 2** para ler a história correspondente.  
3. Para voltar ao menu principal, clique no botão **Voltar** no final da história.  

---


## 🎨 Funcionalidades

- Menu interativo com links para cada história  
- Páginas com textos e imagens centralizadas  
- Navegação simples para voltar ao menu
